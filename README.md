Real Estate Financial & Deductions Dashboard
Project Overview
This project delivers an enterprise-grade financial analytics solution built in **Power BI** to monitor, track, and analyze contractor deductions, penalizations, and overall villa performance metrics across large-scale real estate developments. 
The interactive dashboard empowers project managers and financial controllers to evaluate subcontractor allocations and optimize budget distribution dynamically.
Key Technical Features
Advanced ETL Pipelines: Developed a robust Power Query framework to process, clean, and unpivot a massive multi-column data source containing over 6,900 detailed transaction rows.
Enterprise Data Governance:Applied secure Data Masking techniques directly at the data source level to completely protect sensitive financial records and contractor classifications without affecting data integrity or metric aggregation.
Dynamic Financial Calculations: Implemented custom DAX formulas to instantly calculate Financial Shares (Company Share vs. Subcontractors Share) and dynamic transaction frequencies.
Interactive Data Storytelling: Designed a highly intuitive UI/UX utilizing custom slicers (All Divisions, Creditors, Work Packages), cross-filtering, and historical trend lines over multiple fiscal years.
Business Insights Delivered
1. Total Deductions Analysis: Consolidates multi-million financial impacts in real-time, providing immediate transparency into contractor financial adjustments.
2. Top Penalized Subcontractors: Identifies high-risk vendors and cost centers through Pareto-style distribution charts.
3. Monthly & Yearly Work Trends: Highlights performance seasonality and package distribution (Material, Labor, HSE, Equipment, and Others).
Technologies Used
Power BI Desktop (DAX, Power Query, Data Modeling, UI/UX)
Microsoft Excel (Data preparation, Binary format optimization, Financial accounting structures)
